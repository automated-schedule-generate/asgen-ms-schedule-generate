# microservice do gerador da grade de horarios

## Setup

Instalando dependencias:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```

## Development Server

Por padrão você tem acesso ao servidor via: `http://localhost:9000`:

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev

# bun
bun run dev
```
Obs.: no lugar de dev, pode ser dev:watch para atualizar executar o dev sempre que algum arquivo for alterado

## Production

Gerando build para produção:

```bash
# npm
npm run build

# pnpm
pnpm run build

# yarn
yarn build

# bun
bun run build
```

Servidor executando em produção:

```bash
# npm
npm run start:prod

# pnpm
pnpm run start:prod

# yarn
yarn start:prod

# bun
bun run start:prod
```
